# Blackbar Style Guide  
**Updated for Material UI Icons + Modern MUI Styling**

## Design Philosophy

Blackbar uses a minimal, neutral design system focused on:

- Clean and professional layouts
- Modern Material UI components
- Neutral grays and subtle interaction states
- Strong accessibility
- Predictable structure and theming flexibility

The UI should feel quiet and purposeful. No bright colors, no decorative styles, no unnecessary chrome.
